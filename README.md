# ❤️ HeartRisk EDA — Análise Exploratória de Dados de Risco de Infarto

## 📚 Contexto Acadêmico

Projeto desenvolvido para a disciplina de **Inteligência Artificial & Machine Learning**, com o objetivo de realizar uma **Análise Exploratória de Dados (EDA)** sobre fatores de risco associados a infarto do miocárdio.

## 📊 Dataset

O estudo utiliza um **dataset sintético global** contendo:

- **8.763 registros** de pacientes
- **20 países** representados
- **26 variáveis** clínicas e demográficas, incluindo: idade, sexo, IMC, pressão arterial, colesterol, triglicerídeos, hábitos (tabagismo, consumo de álcool, atividade física), histórico familiar, entre outros

## 🔍 O que o site apresenta

A aplicação web interativa exibe os resultados da análise exploratória através de:

- **Visão geral do dataset** — estrutura, tipos de variáveis e estatísticas descritivas
- **Matriz de correlação** — relações entre variáveis numéricas
- **IMC vs Pressão Arterial** — dispersão entre índice de massa corporal e pressão
- **Idade vs Risco** — distribuição de risco por faixa etária
- **Hábitos de vida** — impacto de tabagismo, álcool e sedentarismo
- **Colesterol e Triglicerídeos** — análise de perfil lipídico
- **Gênero vs Risco** — diferenças de risco entre sexos
- **Conclusões** — principais descobertas da análise
- **Código-fonte** — trechos do código Python utilizado na EDA

## 🛠️ Tecnologias

- **React 18** + **TypeScript**
- **Vite** — build tool
- **Tailwind CSS** — estilização
- **Recharts** — visualização de dados
- **GitHub Pages** — deploy

## 🚀 Como rodar localmente

```bash
# Clonar o repositório
git clone https://github.com/theofilohenrique/heart-risk-insights.git
cd heart-risk-insights

# Instalar dependências
npm install

# Iniciar servidor de desenvolvimento
npm run dev
```

O projeto estará disponível em `http://localhost:5173`.
